**Overview**
- This project is an analysis of the connection between pitching mechanics, pitch types, and pitching injuries. Our goal is to gain a clearer idea of how we can help teams and pitchers stay healthy and prolong their careers.

Team Members
- Johncarlos Fernandez
- Tiffany Munoz
Data Source
- Kaggle: Tommy John Surgery Dataset

Project Goals
- Analyze how pitch type usage relaste to injuries
- Compare different pitch type uasge against injury counts
- Identify If a specific pitch leads to more injuries
